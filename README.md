# Install

**Source**

1. Requirement

```
Go 1.16
```

2. Build binary

```
go get
// Linux
go build -o go-crypto-tracking
// Window
go build
```

3. Run

```
// Linux
./go-crypto-tracking
// Window
.\go-crypto-tracking.exe
```
